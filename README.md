# Kritunga Eats Elevated

Premium dining experience.

## Project Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```

2. Start the development server:
   ```sh
   npm run dev
   ```

## Deployment

Build the project for production:
```sh
npm run build
```
